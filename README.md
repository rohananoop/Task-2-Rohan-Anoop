#  DecodeLabs Java Project - Student Grade Calculator

##  Overview
This is a simple console-based Student Grade Calculator written in Java.  
The program takes number of subjects as input, accepts marks for each subject, calculates total marks, average percentage and grade, and display the result.

## Internship Details
This project was developed as part of a **Virtual Internship Program in Java Programming** at **DecodeLabs**.  
It was **Project 2**, which focus concepts like variables, loops, arithmetic operations, conditional statements.

---

## Features
- Accepts dynamic number of subjects
- Input validation(marks between 0 and 100)
- Calculates total mark and average percentage
- Assigns grade based on average percentage:
  - A (90–100)
  - B (80–89)
  - C (70–79)
  - D (60–69)
  - E (50–59)
  - F (Below 50) 
- Clean formatted result as ouput

---

## How It Works
- User enters number of subjects
- Program inputs marks for each subject
- Invalid marks are rejected and re-entered
- Total marks and average percentage are calculated
- Grade is assigned based on average percentage
